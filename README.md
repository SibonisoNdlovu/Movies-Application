### Features:

- **General**
  - Responsive design.
- **Movies List**
  - List Movies.
  - Search.
- **Movie detail**
  - Show selected movie detail information.

### Prerequisites

To run this project you need:

[NodeJs](https://nodejs.org/en/download/)

### Installing and running

Install the depedencies:

```
npm install
```

Run the application:

```
npm run start
```
